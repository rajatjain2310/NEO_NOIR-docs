Class Stocking Facility :param sku: Unique SKU Identifier (int) :param of_per_week: Hard limit on number of inventory orders to be placed per week (int)

Parameters

  • period – The number of days to run the simulation for (int)

  • ROP_frac – Reorder point trigger based on fraction of current inventory levels (float)

  • main (demand/) – Pandas DataFrame with entire knowledge base of date and units_sold (pd.DataFrame)

  • avg_leadtime – Constant arrival time in days for any inventory order (int)

  • lead_time_var – Variability in historical lead times (std metric) (float)

  • budget_constraint – Inventory order placed on supplier always less than this constraint (float)

  • split_date – Inclusive Date from which the simulation begins (datetime.datetime)

  • invinit – Initial Inventory Definition (int)

  • in_transit – In transit Inventory (int)

  • strategy – Inventory Ordering strategy to be used form XGBOOST, Exp smoothing etc. (str)

  • params – Parameters for MSN in the format [a,b] (array)

  • ordering_freq – Supplier constraints on inventory ordering period (int)

  • price – Price of SKU (int)
